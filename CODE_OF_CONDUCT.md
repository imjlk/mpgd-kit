# Code of Conduct

This project aims to be a focused and respectful place to build game
distribution tooling.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while still being precise about bugs and risks.
- Keep feedback technical, actionable, and relevant to the project.
- Respect maintainer decisions around scope, architecture, and release timing.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Personal attacks or repeated disruptive comments.
- Publishing private information without consent.
- Intentionally derailing technical discussions.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues or pull requests;
or block participants who violate this code of conduct.
